
#Yii Bootstrap Blank Theme#
------------------------------------------------
This is a blank Yii theme for getting you up and runing with Twitter's Bootstarp.
This theme is include bootstarp,font awesome and jquery , so you can use it to develop your website.

----------
![enter image description here][1]

#Installation#
-----------------------

 - Extract files into the Yii application at ***themes/[name of theme]/***

#Usage#


----------

 - add theme info to Yii application main configuation **your application/protected/config/main.php**

 

> return array(
>    'basePath'=>dirname(__FILE__).DIRECTORY_SEPARATOR.'..',
>    'name'=>'My Web Application',
>    'theme'=>'[name of theme]', // THIS LINE HERE
